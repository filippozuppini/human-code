# Human code 

Aprire un pacchetto di figurine

traccia: Che emozione, ogni volta che apro un pacchetto nuovo... chissà se troverò quella figurina così rara!
Dopo averle sfilate dal pacchetto, me le passo tra le mani controllando nell’album: attacco subito quelle nuove, mentre le doppie le metto in un mazzetto a parte. Ormai ne sto accumulando diverse, spero di riuscire a combinare diversi scambi sabato con i miei amici!



- Ricevo il pacchetto
    - SE:
        ricevo il pacchetto
            - ALLORA:
                lo apro
            - ALTRIMENTI:
                non lo apro

- Apro il pacchetto

- Controllo le carte
    - SE:
        riconosco carte doppie
        - ALLORA:
            le metto da parte
    - ALTRIMENTI:
        le tengo in mano

- Apro l'album
    - Controllo se le carte che ho in mano sono giá presenti
    - SE:
        sono giá presenti le metto da parte
    - ALTRIMENTI:
        le attacco

- Vado allo scambio
- Controllo le carte dei miei amici
    - SE:
        trovo carte che non possiedo
    - ALLORA:
        propongo lo scambio con una mia carta
        - SE:
            lo scambio viene accettato 
            - ALLORA:
                ottengo una carta nuova
            - ALTRIMENTI:
                lo scambio viene rifiutato
            - ALLORA:
                non ottengo una carta nuova
    - SE:
        trovo carte che possiedo
        - ALLORA: non propongo lo scambio
